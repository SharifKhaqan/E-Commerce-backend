# E-Commerce Backend

This is the backend for a full-stack E-Commerce project built with Node.js, Express, MongoDB, Stripe payments, and Socket.IO for real-time communication.

---

## How to Run This Project

### 1. Clone the repository
git clone https://github.com/SharifKhaqan/E-Commerce-backend.git

### 2. Go to project folder
cd backend-repo

### 3. Install dependencies
npm install

### 4. Create .env file
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

STRIPE_SECRET_KEY=your_stripe_secret_key

---

### 5. Run the server
npm run dev

OR
npm start

---

## Server will run on:
http://localhost:5000

---


## Notes

- Ensure MongoDB is running or connected via Atlas
